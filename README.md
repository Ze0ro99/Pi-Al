# AiPi Bot
بوت ذكي يدعم مجتمع Pi Network عبر Telegram.

## ميزاته:
- تحليل تلقائي للنصوص والصور.
- تكامل مع Pi SDK و GitHub.
- إرسال ملخصات وتحليلات دورية.
- إعداد تلقائي عبر Termux.

## طريقة التثبيت:
1. شغّل Termux.
2. نفّذ:
```bash
bash setup_and_run.sh
```
3. بعد الإعداد، أضف القيم إلى `.env`.
4. شغّل البوت عبر:
```bash
python main.py
```
